# AMYT

Build macros faster. Record actions, edit scripts, and automate repetitive tasks with a clean desktop workflow.

AMYT is a Windows macro automation app designed for users who want a faster way to create and run powerful scripts without dealing with complicated setup.

## Download

Get the latest Windows installer here:

[Download AMYT](../../releases/latest)

## Why AMYT

AMYT helps you go from idea to working automation quickly.

- Record actions and turn them into script commands
- Build scripts visually with the Command Builder
- Edit and organize everything in a dedicated Script Editor
- Use image, text, and color detection for smarter automation
- Run, pause, and stop scripts with hotkeys
- Manage templates and detection tools in one place
- Focus on speed, usability, and real-world macro workflows

## What You Can Do

With AMYT, you can create automations for tasks like:

- repetitive desktop actions
- game support workflows
- screen-based clicking tasks
- visual detection macros
- keyboard and mouse automation
- multi-step scripted routines

## Built For Ease of Use

AMYT is made for users who want strong automation without a messy experience.

- Clean visual interface
- Script recording and manual editing
- Fast command insertion tools
- Settings for shortcuts, zoom, and user info
- Designed to be practical for everyday use

## Installation

1. Open the latest release
2. Download the installer `.exe`
3. Run the installer
4. Launch AMYT and start building
