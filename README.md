## What Is AMYT?

AMYT is a Windows desktop macro automation app built for users who want speed, control, and a smoother workflow.

It helps you create automations visually, record actions directly from your screen, and fine-tune everything inside a powerful script editor.

This repository is for **public releases only**.  
The source code is not included here.

## Why People Use AMYT

AMYT is designed to make automation easier without losing power.

- Record actions and convert them into usable script commands
- Build scripts visually with the Command Builder
- Edit macros in a dedicated Script Editor
- Use image, text, and color detection
- Run, stop, and pause scripts with hotkeys
- Organize templates and automation tools in one app
- Focus on real workflow speed, not just raw features

---

## Core Features

### Script Recording
Capture your actions and turn them into automation steps quickly.

### Command Builder
Insert commands visually without memorizing syntax.

### Script Editor
Review, format, and manage your scripts in one place.

### Smart Detection Tools
Use templates, OCR text, and color detection for advanced automations.

### Hotkey Controls
Start, stop, and pause scripts instantly.

### Built for Practical Use
Designed for users who want a smooth desktop automation workflow.
